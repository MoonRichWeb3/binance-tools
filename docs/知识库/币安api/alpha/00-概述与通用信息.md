# 00 - 概述与通用信息

来源：

- https://developers.binance.com/docs/zh-CN/alpha/introduction
- https://developers.binance.com/docs/zh-CN/alpha/market-data/general-info

## 功能定位

Binance Alpha 文档提供 Alpha 市场数据能力，当前公开市场数据接口不需要 API Key。它和现货 REST API 不是同一套路径，Alpha REST 使用 Binance Web BAPI：

```text
https://www.binance.com
```

Alpha WebSocket 行情使用：

```text
wss://nbstream.binance.com/w3w/wsa/stream
```

## 通用规则

| 项目 | 规则 |
| --- | --- |
| 认证 | 公共市场数据接口不需要身份验证 |
| REST 格式 | `GET` 请求，返回 JSON |
| 成功码 | `code = "000000"` |
| 成功标记 | 多数接口包含 `success = true` |
| 时间戳 | 毫秒 |
| symbol | 使用 Alpha 格式，例如 `ALPHA_175USDT` |

## Alpha Symbol 构造

Alpha REST 文档强调：调用行情接口前，先通过 Token 列表获取 `alphaId`。例如 Token 列表返回：

```text
alphaId = ALPHA_175
symbol = gorilla
```

如果计价资产为 `USDT`，交易对 symbol 示例为：

```text
ALPHA_175USDT
```
