# GPUI Component 导读：安装与入门

> 内容依据官网 [Installation](https://longbridge.github.io/gpui-component/docs/installation)、[Getting Started](https://longbridge.github.io/gpui-component/docs/getting-started) 与 [Introduction](https://longbridge.github.io/gpui-component/docs/) 整理为中文摘要；细节与 API 以官网及本地 [`llms-full.txt`](./llms-full.txt) 为准。

## 1. 是什么

**GPUI Component** 是基于 [GPUI](https://gpui.rs/) 的 Rust 桌面 UI 组件库，用于构建跨平台原生风格应用。官方文档入口：[GPUI Component 文档](https://longbridge.github.io/gpui-component/docs/)。

## 2. 系统与工具链要求（摘要）

- **macOS**：macOS 15+，需 Xcode Command Line Tools。
- **Windows**：Windows 10+；可用仓库内 `script\install-window.ps1`（PowerShell）辅助安装依赖。
- **Linux**：可运行 `./script/bootstrap` 安装系统依赖。
- **Rust**：文档当前要求 **Rust 1.90+**（随官方更新可能变化，请以安装页为准）。

## 3. `Cargo.toml` 依赖（典型写法）

```toml
[dependencies]
gpui = { git = "https://github.com/zed-industries/zed" }
gpui_platform = { git = "https://github.com/zed-industries/zed", features = ["font-kit"] }
gpui-component = { git = "https://github.com/longbridge/gpui-component" }
# 可选：默认图标等资源
gpui-component-assets = { git = "https://github.com/longbridge/gpui-component" }
anyhow = "1.0"
```

- **`gpui-component-assets`**：可选；提供默认图标资源。若自建资源管理，可不添加（详见官方 [Icons & Assets](https://longbridge.github.io/gpui-component/docs/assets)）。

## 4. 最小可运行结构要点

1. 在 `app.run` 闭包**开头**调用 **`gpui_component::init(cx)`**，用于主题等全局初始化。
2. 使用 **`gpui_platform::application()`**；若使用默认资源，可用 **`.with_assets(gpui_component_assets::Assets)`**。
3. 窗口根视图应包在 **`Root::new(view, window, cx)`** 中（官方 Quick Start 示例）。

示例代码请直接对照官网 *Getting Started* 或本地 `llms-full.txt` 中 `getting-started.md` 一节。

## 5. 核心概念（摘要）

### 无状态与 `IntoElement`

多数组件为无状态的 **`RenderOnce`** 风格元素，实现 **`IntoElement`**；状态放在 **View** 层，由你在 `Render` 实现里组合子组件。

### 有状态组件

如 **`Dropdown`**、**`List`**、**`Table`** 等内部自管状态，实现 **`Render`**；需在 View 结构体中持有 **`Entity<...State>`** 并在 `cx.new` 中创建（见官方示例）。

### 主题

通过内置 **`Theme`** / **`ActiveTheme`** 使用，例如 `cx.theme().primary`、`cx.theme().background` 等。

### 尺寸与变体

- 尺寸：如 `Button::new("id").small()` / `.medium()` / `.large()` / `.xsmall()`。
- 变体：如 `.primary()`、`.danger()`、`.warning()`、`.success()`、`.ghost()`、`.outline()` 等。

### 图标

组件库为减轻体积**不内置 SVG**；通过 **`Icon::new(IconName::...)`** 使用，示例常用 [Lucide](https://lucide.dev/)，也可按 `IconName` 约定挂载自有资源（见官方 *Icons & Assets*）。

## 6. 本地开发与示例

- 运行组件画廊（在 **gpui-component** 仓库根目录）：`cargo run`。
- 运行单个示例：`cargo run --example <名称>`。
