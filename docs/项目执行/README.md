## 当前主功能

- [✓] 【完成】 Dashboard 首页展示 Binance Web product 市场榜单，数据缓存到 SQLite 5 分钟。
- [✓] 【完成】 现货菜单包含市场榜单、币种列表、市场热力图、日均线信号、K 线图和现货回测。
- [✓] 【完成】 现货币种列表支持搜索、按 Quote 计价资产切换，并统计上涨、下跌、持平数量。
- [✓] 【完成】 市场热力图按市值或成交额权重展示涨跌色块，支持搜索、Quote 分类和点击进入 K 线。
- [✓] 【完成】 币安广场菜单包含 Key 设置、任务页面和发送消息日志。
- [✓] 【完成】 币安广场 AI 生成每日最多 100 条，失败最多重试 3 次且间隔 5 分钟。
- [✓] 【完成】 币安广场 AI 规则和兜底文案从 `ai_rules` 表读取，不再在业务流程里写死。
- [✓] 【完成】 右侧 AI Agent 面板支持 OpenAI-compatible provider 配置、模型选择、流式输出、重试、继续和复制。
- [✓] 【完成】 右侧 AI Agent 面板支持 Zed 风格历史会话列表和会话恢复。
- [✓] 【完成】 页面级 AI 规则保存到 SQLite `ai_rules` 表，按页面独立读取并压缩存储。
- [✓] 【完成】 `Rules` 独立弹窗支持新增、搜索、编辑、Markdown/Text 格式选择、眼睛图标预览、保存、复制、刷新和启用/禁用规则。
- [✓] 【完成】 AI 历史会话保存规则快照，后续修改规则不会覆盖旧分析记录。
- [✓] 【完成】 市场榜单可点击 `AI 分析`，把当前筛选市场前 50 条精简数据发送到右侧 AI 面板分析。
- [✓] 【完成】 Binance Alpha 官方文档已入库，REST 市场数据接口、WebSocket stream 辅助和顶部 Alpha 菜单已实现。
- [✓] 【完成】 Alpha Token 列表支持链分类、搜索、积分+、证券代币分类、中文表头、合约复制和 K 线入口。
- [✓] 【完成】 Alpha 交易对页面支持搜索、Quote 分类、涨跌统计和本地 SQLite 缓存。
- [✓] 【完成】 Alpha 日均线和 Alpha K 线读取 `alpha_klines` 缓存，缺失时自动补齐。
- [✓] 【完成】 Alpha 热力图复用现货热力图交互，按市值或 24h 成交额权重展示。
- [✓] 【完成】 工具菜单包含文档转换，支持 Markdown / HTML 本地互转、复制和保存。
- [✓] 【完成】 工具菜单包含任务看板，支持卡片、弹窗新增/修改、详情查看、提醒高亮、完成/恢复/删除。
- [✓] 【完成】 帮助菜单包含策略说明，可选择每个回测策略并以 Markdown 格式查看创始人、计算公式、适合行情和历史故事。
- [✓] 【完成】 现货回测从 Binance Vision 下载缺失 K 线，本地缓存解压 CSV，后续只补缺失日期。
- [✓] 【完成】 现货回测支持图形化 K 线、成交量、EMA、买卖标记、拖动、缩放和回测指标。
- [✓] 【完成】 回测策略支持 `MA Cross`、`Grid`、`Trend Grid`、`Turtle`、`Martingale`、`RSI`、`MACD`、`Bollinger Bands`、`Volume Spike`、`OBV`、`Stochastic`、`CCI`、`SuperTrend`，每个策略单独一个文件。

## 维护约定

- 新增页面或跨模块功能时，在本目录增加或更新对应文档。
- 数据库表结构请同步更新 [`../db_table`](../db_table/README.md)。
- 文档中的路径尽量写到具体模块，避免只描述 UI 截图。
- 已经完成并落地到代码的功能项统一用 `[✓] 【完成】` 标记。
- 尚未实现或只保留占位的功能统一用 `x 【未完成】` 标记。
