# 10 项目工具计算器

本文记录桌面端 `工具 -> 计算器` 功能。

## 入口

```text
工具 -> 计算器
```

## 实现位置

| 模块 | 路径 | 说明 |
| --- | --- | --- |
| 菜单入口 | `examples/desktop-gpui/src/ui/title_bar.rs` | 新增 `OpenCalculator` action 和菜单项 |
| 页面切换 | `examples/desktop-gpui/src/ui/dashboard.rs` | 打开 `CalculatorPage` |
| 计算器页面 | `examples/desktop-gpui/src/ui/calculator.rs` | 计算器按钮、显示区、角度模式切换 |
| 计算核心 | `src/calculator/mod.rs` | 表达式解析、函数计算、历史答案 `Ans` |

## 功能清单

- [✓] 【完成】 计算器代码拆成独立 `src/calculator` 模块，UI 不直接写计算规则。
- [✓] 【完成】 支持数字、小数点、括号和 `+ - × ÷` 四则运算。
- [✓] 【完成】 支持 `x^y` 幂运算、`√` 平方根、`x!` 阶乘、`%` 百分号。
- [✓] 【完成】 支持 `sin`、`cos`、`tan`、`ln`、`log`。
- [✓] 【完成】 支持 `Inv`，反向切换到 `asin`、`acos`、`atan`、`e^x`、`10^x`、平方。
- [✓] 【完成】 支持 `Deg / Rad` 角度模式。
- [✓] 【完成】 支持 `π`、`e`、`EXP` 科学计数和 `Ans` 上次答案。
- [✓] 【完成】 支持 `AC` 清空和 `=` 计算。

## 设计说明

计算核心使用递归下降解析器：

- 先解析括号、数字、常量和函数。
- 再处理阶乘、百分号等后缀。
- 然后处理幂运算。
- 最后处理乘除和加减。

这样做的好处是表达式优先级明确，比如 `2 + 3 × 4` 会先算乘法，再算加法。
