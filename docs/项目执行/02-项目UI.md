# 02 项目 UI

本文说明 `examples/desktop-gpui/` 的当前定位、页面结构和 UI 约定。

## 定位

`desktop-gpui` 是 `binance-tools` 的桌面 UI 工程。业务数据读取、Binance 请求、SQLite 读写和 AI 请求逻辑优先沉淀在根 crate `src/`，UI 层只负责状态、交互和渲染。

| 项 | 当前实现 |
| --- | --- |
| 工程路径 | `examples/desktop-gpui/` |
| Cargo 包名 | `desktop-gpui` |
| 运行方式 | `cargo run -p desktop-gpui` |
| UI 技术 | `gpui` + `gpui-component` |
| 业务依赖 | `binance_tools = { path = "../.." }` |
| 默认窗口 | 启动时居中打开，默认尺寸 `1024 x 768` |
| Windows 图标 | `assets/app.ico`，通过 `build.rs` 和 `winresource` 嵌入 exe |

标题栏当前菜单：

| 菜单 | 说明 |
| --- | --- |
| `主题` | 切换 `themes/*.json` 中注册的主题，并保存到 `config/setings.json` |
| `现货` | 进入市场榜单、币种、日均线信号 |

## 布局约定

- 主窗口由标题栏、左侧内容区、可拖拽分隔线和右侧 AI 面板组成。
- 右侧 AI 面板宽度可通过分隔线拖动。
- AI Settings 支持在右侧面板内打开，也支持最大化覆盖主内容区。
- 启动窗口尺寸在 `src/app.rs` 的 `WindowOptions.window_bounds` 中设置。
- 所有表格页面外层使用完整高度，表格区域随窗口高度伸缩。
- 表格统一使用 `DataTable`，开启条纹、边框和滚动条。
- 按钮尺寸统一使用 `xsmall()`。
- 文本颜色通过 `src/ui/palette.rs` 适配当前主题，避免固定高对比硬编码。
- 市场榜单头部展示当前筛选市场的上涨、下跌和持平数量；搜索框支持按交易对、币种名称、分区和标签匹配，统计基于搜索后的当前可见交易对。
- 日均线信号打开的 K 线图使用弹性图表面板，窗口放大时价格 K 线区域跟随可用空间扩展。
- 市场榜单和日均线信号的币种列在币名后提供 K 线图标按钮，点击后进入对应交易对的 K 线页面。

## 图标与打包

- `examples/desktop-gpui/assets/app.ico` 是当前应用图标，设计为任务栏小尺寸下可识别的放大猫图标。
- `examples/desktop-gpui/build.rs` 仅在 Windows 编译时调用 `winresource`，把 `app.ico` 写入 exe 资源。
- `Cargo.toml` 中的 `[package.metadata.packager]` 已配置 `product-name`、`identifier` 和 `icons`，后续可直接给 `cargo-packager` 复用。
- 替换图标时保持路径 `assets/app.ico` 不变，然后重新执行 `cargo build -p desktop-gpui --release`。

## 主题

主题由 `examples/desktop-gpui/src/theme.rs` 管理，标题栏 `主题` 菜单读取 `themes/*.json`。默认主题为 `Ayu Light`，用户选择会保存到 `config/setings.json`，下次启动时自动恢复。UI 颜色应从当前 `Theme` 和 `palette` 读取。

## 用户配置

桌面 UI 的用户偏好配置位于：

```text
config/setings.json
```

当前字段：

| 字段 | 说明 |
| --- | --- |
| `theme` | 用户最后一次在标题栏 `主题` 菜单中选择的主题名称 |

如果文件不存在，启动时会按默认主题 `Ayu Light` 创建。用户每次切换主题都会立即写回该文件。

## 业务边界

| 层级 | 职责 |
| --- | --- |
| `examples/desktop-gpui/src/ui/` | 页面布局、组件状态、用户交互、菜单和错误展示 |
| `src/binance/` | Binance REST/Web 数据结构和接口调用 |
| `src/db/` | SQLite 建表、缓存、查询和更新 |
| `src/ai/` | AI 配置、Provider 选择、HTTP 请求和流式解析 |
| `src/square/` | 币安广场发送执行器和任务调度相关业务逻辑 |

UI 层不直接拼接 SQL，也不直接处理 Binance SDK 原始响应。
