# 09 - 现货回测

> 本文记录通过 Binance Vision 公共数据实现现货回测页面的当前完成状态。

## 数据来源

- [✓] 【完成】 使用 Binance Vision 现货公共数据：`https://data.binance.vision/?prefix=data/spot/`。
- [✓] 【完成】 K 线 zip 路径规则：`/data/spot/{daily|monthly}/klines/{SYMBOL}/{INTERVAL}/{SYMBOL}-{INTERVAL}-{YYYY-MM-DD}.zip`。
- [✓] 【完成】 本地回测数据包缓存：解压后的 CSV 保存到 `data/backtest/spot/daily/klines/{SYMBOL}/{INTERVAL}/`。
- [✓] 【完成】 回测前优先读取本地缓存，只下载本地缺少的日期文件，避免每次重复下载。
- [✓] 【完成】 回测 K 线会写入 SQLite，后续回测优先按 `symbol + interval + open_time` 从数据库读取。
- [✓] 【完成】 `4h` 回测数据单独写入 `spot_klines_4h` 表。
- [✓] 【完成】 `1d` 回测数据单独写入 `spot_klines_1d` 表。
- [✓] 【完成】 旧 `spot_klines` 中已有的 `4h` / `1d` 数据会在迁移时复制到对应分表。
- [✓] 【完成】 Kline CSV 按 Binance Spot `/api/v3/klines` 字段解析：开盘时间、开高低收、成交量、收盘时间等。
- [✓] 【完成】 2025-01-01 之后 spot 文件可能使用微秒时间戳，解析层已自动转换为毫秒。
- [✓] 【完成】 单次下载限制为最多 366 天，避免误操作导致大量请求。

- [✓] 【完成】 支持输入交易对，例如 `BTCUSDT`。
- [✓] 【完成】 支持输入周期，默认 `4h`，也支持 `1d` 等 Binance Vision Kline interval。

- [✓] 【完成】 默认开始日期为 `2026-01-01`。
- [✓] 【完成】 支持输入开始日期、结束日期，格式为 `YYYY-MM-DD`。
- [✓] 【完成】 支持输入初始资金和手续费率。
- [✓] 【完成】 策略下拉框根据当前策略切换参数输入项，并填入默认参数。
- [✓] 【完成】 回测结果展示 K 线条数、缓存文件数、下载文件数、缺失文件数、初始资金、最终权益、收益率、最大回撤、交易次数、胜率。
- [✓] 【完成】 展示交易记录：时间、买入/卖出、价格、数量、权益。

## 策略能力

- [✓] 【完成】 `MA Cross`：短均线上穿长均线买入，短均线下穿长均线卖出。
- [✓] 【完成】 `Grid`：支持网格下限、网格上限、网格数量参数，价格下穿网格买入，上穿网格卖出。
- [✓] 【完成】 `Trend Grid`：价格在趋势 EMA 上方才允许买入，跌破 EMA 且超过止损比例时清仓。
- [✓] 【完成】 `Turtle`：突破周期入场、退出周期或 ATR 止损离场，支持单次仓位比例。
- [✓] 【完成】 `Martingale`：下跌按倍数加仓，达到止盈比例后清仓，支持最大层数限制。
- [✓] 【完成】 `RSI`：RSI 低于超卖阈值买入，高于超买阈值或触发止损时卖出，支持周期、阈值、仓位和止损参数。
- [✓] 【完成】 `MACD`：MACD 快慢线金叉且位于零轴上方买入，死叉或触发止损时卖出。
- [✓] 【完成】 `Bollinger Bands`：价格触及下轨买入，回到中轨、达到止盈或触发止损时卖出。
- [✓] 【完成】 `Volume Spike`：价格突破前高且成交量达到放量倍数时买入，跌破价格均线或触发止损时卖出。
- [✓] 【完成】 `OBV`：OBV 上穿均线且价格站上价格均线时买入，OBV 下穿、止盈或止损时卖出。
- [✓] 【完成】 `Stochastic`：低位 %K 上穿 %D 买入，高位 %K 下穿 %D 卖出。
- [✓] 【完成】 `CCI`：CCI 进入超卖区买入，进入超买区或触发止损时卖出。
- [✓] 【完成】 `SuperTrend`：趋势翻多买入，趋势翻空、止盈或止损时卖出。
- [✓] 【完成】 回测算法层增加统一 `run_backtest` 入口，UI 不直接调用具体策略函数。
- [✓] 【完成】 每个策略单独文件维护，后续新增策略只需要增加策略文件并接入 `BacktestStrategy`。
