# 09 - 现货回测

> 本文记录通过 Binance Vision 公共数据实现现货回测页面的当前完成状态。

## 数据来源

- [x] 【完成】 使用 Binance Vision 现货公共数据：`https://data.binance.vision/?prefix=data/spot/`
- [x] 【完成】 日线 zip 路径规则：`/data/spot/daily/klines/{SYMBOL}/{INTERVAL}/{SYMBOL}-{INTERVAL}-{YYYY-MM-DD}.zip`
- [x] 【完成】 增加本地回测数据包缓存：解压后的 CSV 保存到 `data/backtest/spot/daily/klines/{SYMBOL}/{INTERVAL}/`。
- [x] 【完成】 回测前优先读取本地缓存，只下载本地缺少的日期文件，避免每次重复下载。
- [x] 【完成】 Kline CSV 按 Binance Spot `/api/v3/klines` 字段解析：开盘时间、开高低收、成交量、收盘时间等。
- [x] 【完成】 2025-01-01 之后 spot 文件可能使用微秒时间戳，解析层已自动转换为毫秒。

## 程序结构

- [x] 【完成】 `src/binance/vision.rs`：负责从 Binance Vision 下载现货 daily klines zip，并解析为 `VisionKline`。
- [x] 【完成】 `src/backtest.rs`：负责回测计算，不绑定 UI。
- [x] 【完成】 `examples/desktop-gpui/src/ui/backtest.rs`：桌面端现货回测页面。
- [x] 【完成】 `examples/desktop-gpui/src/ui/title_bar.rs`：在 `现货` 菜单中增加 `回测` 入口。
- [x] 【完成】 `examples/desktop-gpui/src/ui/dashboard.rs`：将回测页面接入主内容区。

## 当前回测能力

- [x] 【完成】 支持输入交易对，例如 `BTCUSDT`。
- [x] 【完成】 支持输入周期，第一版默认 `1d`。
- [x] 【完成】 周期增加 Binance Vision 支持值校验：`1s`、`1m`、`3m`、`5m`、`15m`、`30m`、`1h`、`2h`、`4h`、`6h`、`8h`、`12h`、`1d`、`3d`、`1w`、`1mo`。
- [x] 【完成】 页面提示 `120d` 不是 Binance Kline interval；120 日回测应使用 `1d`，并选择 120 天日期范围。
- [x] 【完成】 默认开始日期改为当前日期前 120 天，降低误填周期的概率。
- [x] 【完成】 支持输入开始日期、结束日期，格式为 `YYYY-MM-DD`。
- [x] 【完成】 支持输入短均线、长均线、初始资金、手续费率。
- [x] 【完成】 第一版策略为 MA Cross：短均线上穿长均线买入，短均线下穿长均线卖出。
- [x] 【完成】 回测结果展示 K 线条数、下载文件数、缺失文件数、初始资金、最终权益、收益率、最大回撤、交易次数、胜率。
- [x] 【完成】 回测结果展示缓存命中文件数，用于区分本地读取和本次补齐下载。
- [x] 【完成】 回测结果增加图形化 K 线走势面板，复用 gpui-component `CandlestickChart`。
- [x] 【完成】 回测 K 线图支持类似 TradingView 的鼠标拖动平移。
- [x] 【完成】 回测 K 线图支持滚轮缩放，并提供 `+` / `-` 按钮调整可见 K 线数量。
- [x] 【完成】 回测页面调整为图表优先布局：压缩顶部表单和指标卡片占比，放大 K 线主图和成交量区。
- [x] 【完成】 K 线图下方增加成交量柱状图，复用 gpui-component `BarChart`。
- [x] 【完成】 回测交易点在 K 线图上叠加买入/卖出标记，便于直接观察策略进出场位置。
- [x] 【完成】 展示交易记录：时间、买入/卖出、价格、数量、权益。
- [x] 【完成】 单次下载限制为最多 366 天，避免误操作导致大量请求。
