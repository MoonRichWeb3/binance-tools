# 08 工具模块

本文记录桌面端 `工具` 菜单下的本地工具页面。

## 文档转换

入口：

```text
工具 -> 文档转换
```

实现位置：

| 模块 | 路径 | 说明 |
| --- | --- | --- |
| 菜单入口 | `examples/desktop-gpui/src/ui/title_bar.rs` | `工具` 菜单和 `文档转换` action |
| 页面切换 | `examples/desktop-gpui/src/ui/dashboard.rs` | 打开 `DocumentConvertPage` |
| 转换页面 | `examples/desktop-gpui/src/ui/tools.rs` | 文件打开、转换、复制、保存 |

功能清单：

- [✓] 【完成】 支持打开本地 Markdown / HTML 文件。
- [✓] 【完成】 支持直接粘贴源内容。
- [✓] 【完成】 支持 `MD -> HTML`。
- [✓] 【完成】 支持 `HTML -> MD` 基础转换。
- [✓] 【完成】 支持复制转换结果。
- [✓] 【完成】 支持通过系统保存对话框保存转换结果。
- [✓] 【完成】 Markdown 转 HTML 使用 `pulldown-cmark`，结果导出为完整 HTML 文档。
- [✓] 【完成】 打开 `.html` / `.htm` 文件时默认切到 `HTML -> MD`，其它文件默认 `MD -> HTML`。

## 任务看板

入口：

```text
工具 -> 任务看板
```

实现位置：

| 模块 | 路径 | 说明 |
| --- | --- | --- |
| 菜单入口 | `examples/desktop-gpui/src/ui/title_bar.rs` | `任务看板` action |
| 页面切换 | `examples/desktop-gpui/src/ui/dashboard.rs` | 打开 `TaskBoardPage` |
| 看板页面 | `examples/desktop-gpui/src/ui/task_board.rs` | 卡片列表、表单、提醒状态 |
| 数据库 | `src/db/task_board.rs` | `tool_board_tasks` 表读写 |

功能清单：

- [✓] 【完成】 支持新增任务标题、备注和提醒时间。
- [✓] 【完成】 页面顶部加号打开新增任务弹窗。
- [✓] 【完成】 任务以卡片形式展示，页面主体只显示卡片内容。
- [✓] 【完成】 点击任务卡片打开详情弹窗，可查看完整内容。
- [✓] 【完成】 详情弹窗支持修改任务。
- [✓] 【完成】 未完成任务到提醒时间后顶部提示并高亮卡片。
- [✓] 【完成】 临近 30 分钟内的任务显示临近状态。
- [✓] 【完成】 支持完成、恢复和删除任务。
- [✓] 【完成】 任务保存到 SQLite `tool_board_tasks` 表。
- [✓] 【完成】 页面每 30 秒刷新一次提醒状态。

## 已知边界

- x 【未完成】 `HTML -> MD` 当前是基础转换，适合常见标题、段落、换行、列表和实体反转；复杂 HTML 表格、嵌套样式、脚本和内联 CSS 不做完整语义还原。
- x 【未完成】 文档转换当前不写入数据库，转换内容只在当前页面内存中保存。
- x 【未完成】 任务看板提醒目前是应用内提醒，不调用系统通知中心；需要打开看板页面才能看到高亮。
