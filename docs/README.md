# 文档目录

本目录保存项目设计、执行记录、数据库表结构和外部知识库资料。当前代码功能以 `src/` 和 `examples/desktop-gpui/` 为准，项目自有功能说明主要维护在 `docs/项目执行/` 和 `docs/db_table/`。

## 目录

| 路径 | 说明 |
| --- | --- |
| [`项目执行/`](./项目执行/README.md) | 项目框架、桌面 UI、币安现货、币安广场、AI 模块和市场榜单 AI 分析说明 |
| [`安装/`](./安装/README.md) | 本地运行、Windows Release 构建和安装包制作说明 |
| [`db_table/`](./db_table/README.md) | SQLite 表结构和字段说明 |
| [`知识库/`](./知识库/README.md) | Binance API、SDK、GPUI Component 等外部资料摘录和索引 |
| [`策略知识/`](./策略知识/) | 技术指标、策略基础和 K 线知识 |

## 维护约定

- 功能代码变更后，同步更新 `项目执行/` 中对应文档。
- SQLite 表结构变更后，同步更新 `db_table/` 中对应文档。
- `知识库/` 主要保存外部参考资料，不要求和当前代码功能一一对应。
- 运行时生成的 SQLite 数据库、缓存和日志不写入文档目录。
