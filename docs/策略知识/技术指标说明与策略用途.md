## 1. 使用说明

本文用于解释项目中计划支持的技术指标，包括它们的基本含义、计算原理、历史渊源、常见用途、常见策略和项目内重要程度。

重要程度采用 5 星制：

| 星级 | 含义 |
| --- | --- |
| ★★★★★ | V1 核心指标，页面和详情页优先实现 |
| ★★★★☆ | 高频使用指标，适合 V1 后半段或 V2 策略优先接入 |
| ★★★☆☆ | 有价值但不宜单独决策，适合作为辅助过滤 |
| ★★☆☆☆ | 特定场景有效，默认不作为主指标 |
| ★☆☆☆☆ | 暂不建议优先实现 |

技术指标不是预测机器。它们只能把历史价格、成交量和波动结构转换成更容易观察的形态。尤其在加密货币市场，假突破、高波动、插针和流动性变化很常见，任何指标都应结合趋势、成交量、市场环境和风险控制使用。

## 3. SMA / MA 简单移动平均线

### 解释

SMA 是 Simple Moving Average，中文常称简单移动平均线。它取最近 N 根 K 线的价格平均值，并随着新 K 线出现不断向前移动。

例如 20 日 SMA 表示最近 20 个交易周期收盘价的平均值。

### 原理

```text
SMA = 最近 N 个收盘价之和 / N
```

SMA 对窗口内每个价格赋予相同权重，因此稳定、平滑，但反应偏慢。

### 用途

1. 判断趋势方向：均线向上表示趋势偏强，均线向下表示趋势偏弱。
2. 判断价格相对位置：价格在均线上方通常偏强，在均线下方通常偏弱。
3. 判断支撑压力：中长期均线常被市场参与者视作动态支撑或压力。
4. 判断均线排列：短期均线在长期均线上方，代表偏多结构；反之偏空。

### 历史渊源

移动平均来自统计中的时间序列平滑思想，后来成为技术分析中最基础的趋势工具。Britannica Money 将移动平均定义为对连续价格按时间窗口求平均，用于平滑短期波动和观察趋势。

### 常用策略

1. 均线突破：价格上穿重要均线视为趋势转强。
2. 均线交叉：短均线上穿长均线为金叉，下穿为死叉。
3. 多头排列：短、中、长期均线依次向上排列，作为趋势确认。
4. 回踩均线：趋势中价格回踩均线不破，作为顺势入场观察点。

### 局限

SMA 是滞后指标，震荡行情中容易频繁假突破和假交叉。

### 项目建议

重要程度：★★★★★

V1 必做。列表页中的周、月、季度、4 月、半年均线都属于 MA 体系，应优先保证计算口径一致。

## 4. EMA 指数移动平均线

### 解释

EMA 是 Exponential Moving Average，中文常称指数移动平均线。它也是移动平均线，但对最近价格赋予更高权重。

### 原理

EMA 使用指数平滑思想，越新的价格权重越高，越旧的价格权重越低。

常见平滑系数：

```text
alpha = 2 / (N + 1)
EMA_today = Close_today * alpha + EMA_yesterday * (1 - alpha)
```

### 用途

1. 捕捉短线趋势变化。
2. 作为 MACD 等指标的基础。
3. 与 SMA 搭配，区分短线动量和长期趋势。

### 历史渊源

EMA 来自指数平滑方法，在金融技术分析中被广泛用于趋势跟踪。Britannica Money 对比 SMA 与 EMA 时指出，EMA 会给近期价格更高权重，因此更适合短期交易场景。

### 常用策略

1. 快慢 EMA 交叉。
2. 价格回踩 EMA 后继续顺趋势运行。
3. EMA 多周期排列，例如 9/21/55 EMA。
4. EMA 与 MACD 联合判断动量。

### 局限

EMA 更灵敏，也更容易在高噪声市场中产生误导信号。

### 项目建议

重要程度：★★★★★

V1 必做，尤其适合详情页叠加在 K 线上；列表页可以优先展示 SMA/MA，详情页提供 EMA 切换。

## 5. 周/月/季度/4月/半年均线

### 解释

这是项目 V1 列表页的核心展示指标，用于紧凑表达一个币种在多个中长期周期上的相对结构。

建议展示列名：

```text
周/月/季/4月/半年
```

### 原理

可以有两种计算口径：

1. 基于日 K 近似：周=7 日、月=30 日、季度=90 日、4 月=120 日、半年=180 日。
2. 基于周期 K 线：按周 K、月 K、季度 K 等重采样后计算均线。

V1 推荐先使用日 K 近似，原因是实现简单、可比较性强、查询成本低。后续如果需要更符合传统周期图表，可再加入周期 K 线口径。

### 用途

1. 快速判断币种处在短中长期多头还是空头结构。
2. 判断当前价格相对中长期成本区的位置。
3. 列表页快速筛选强势币、弱势币、趋势反转候选币。

### 历史渊源

该组合不是单一经典指标，而是移动平均线在多周期趋势分析中的项目化应用。其理论基础来自移动平均的趋势平滑用途。

### 常用策略

1. 多周期同向：价格同时站上周/月/季/半年均线，筛选强趋势资产。
2. 短强长弱：价格站上周/月均线但仍低于季度/半年均线，观察中期反转。
3. 长强短弱：价格仍在季度/半年均线上方但跌破周/月均线，观察回调风险。
4. 多周期压制：全部低于均线，默认不做多或只观察反弹。

### 局限

周期越长越滞后；加密货币新币历史不足时，季度、4 月、半年均线可能不可用。

### 项目建议

重要程度：★★★★★

V1 必做。列表页优先显示状态而非冗长数值，例如：

```text
上/上/下/下/下
```

详情页再展示具体均线值。

## 6. MACD

### 解释

MACD 是 Moving Average Convergence Divergence，中文常称指数平滑异同移动平均线。它用于观察趋势方向、趋势动量和趋势加速/减速。

### 原理

常见参数为 12、26、9：

```text
MACD Line = EMA(12) - EMA(26)
Signal Line = EMA(MACD Line, 9)
Histogram = MACD Line - Signal Line
```

### 用途

1. 趋势确认。
2. 动量增强或减弱判断。
3. 金叉死叉。
4. 顶背离和底背离。

### 历史渊源

MACD 通常归功于 Gerald Appel，资料普遍记载其在 1970 年代后期提出。Britannica Money 也将 MACD 描述为 Appel 开发的趋势和动量识别工具。

### 常用策略

1. MACD 金叉：MACD Line 上穿 Signal Line，偏多。
2. MACD 死叉：MACD Line 下穿 Signal Line，偏空。
3. 零轴上方金叉：比零轴下方金叉更偏趋势延续。
4. 背离：价格创新高但 MACD 不创新高，警惕动量衰减。
5. 柱状图收缩：趋势动能开始减弱。

### 局限

MACD 本质仍基于均线，存在滞后；在横盘震荡中容易出现多次无效交叉。

### 项目建议

重要程度：★★★★★

V1 必做，详情页必须展示。V2 策略中可作为趋势确认和共振评分的重要组成。

## 7. RSI

### 解释

RSI 是 Relative Strength Index，中文常称相对强弱指数。它衡量一段时间内上涨幅度和下跌幅度的相对关系，通常用于判断动量强弱、超买超卖和背离。

### 原理

常见周期为 14：

```text
RS = 平均上涨幅度 / 平均下跌幅度
RSI = 100 - 100 / (1 + RS)
```

RSI 取值范围为 0 到 100。

常见阈值：

```text
RSI > 70：可能超买
RSI < 30：可能超卖
```

### 用途

1. 判断超买超卖。
2. 观察动量变化。
3. 寻找顶背离和底背离。
4. 判断震荡区间中的反转点。

### 历史渊源

RSI 由 J. Welles Wilder Jr. 提出，并发表于 1978 年的 *New Concepts in Technical Trading Systems*。该书也介绍了 ATR 等经典指标。

### 常用策略

1. RSI 超卖反弹：RSI 从 30 下方回到 30 上方。
2. RSI 超买回落：RSI 从 70 上方跌回 70 下方。
3. RSI 背离：价格和 RSI 方向不一致。
4. 趋势过滤：强趋势中 RSI 可能长时间处于高位或低位，因此需要结合趋势指标。

### 局限

RSI 在强趋势中可能长时间超买或超卖，过早逆势交易风险很大。

### 项目建议

重要程度：★★★★★

V1 必做。列表页可展示 RSI 当前值和状态；详情页用于配合 K 线和 MACD 分析。

## 8. Bollinger Bands 布林带

### 解释

Bollinger Bands 由中轨、上轨和下轨组成。中轨通常是 20 周期 SMA，上下轨通常是中轨加减 2 倍标准差。

### 原理

```text
Middle Band = SMA(20)
Upper Band = SMA(20) + 2 * Standard Deviation(20)
Lower Band = SMA(20) - 2 * Standard Deviation(20)
```

标准差越大，带宽越宽；标准差越小，带宽越窄。

### 用途

1. 判断波动率变化。
2. 观察价格偏离均值程度。
3. 识别布林带收缩后的潜在突破。
4. 辅助判断通道上沿和下沿。

### 历史渊源

Bollinger Bands 由 John Bollinger 在 1980 年代早期提出。Britannica Money 将其描述为结合移动平均和波动率度量的技术分析工具。

### 常用策略

1. 布林带挤压：带宽收窄后等待放量突破。
2. 均值回归：震荡行情中触及上轨警惕回落，触及下轨观察反弹。
3. 趋势贴边：强趋势中价格可能沿上轨或下轨运行，不应简单逆势。
4. 与成交量结合：突破布林带并放量，比无量突破更可信。

### 局限

布林带不能直接告诉方向。低波动后可能向上突破，也可能向下突破。

### 项目建议

重要程度：★★★★★

V1 必做。详情页必须展示；列表页可展示带宽、价格位置或是否处于挤压状态。

## 9. ATR 平均真实波幅

### 解释

ATR 是 Average True Range，中文常称平均真实波幅。它衡量市场波动程度，而不是判断涨跌方向。

### 原理

先计算 True Range：

```text
TR = max(
  High - Low,
  abs(High - Previous Close),
  abs(Low - Previous Close)
)
ATR = TR 的 Wilder 平滑均值
```

### 用途

1. 衡量波动率。
2. 设置止损距离。
3. 根据波动调整仓位。
4. 判断市场是否进入高波动或低波动状态。

### 历史渊源

ATR 由 J. Welles Wilder Jr. 在 1978 年提出，常被归入其 *New Concepts in Technical Trading Systems* 中的核心指标体系。

### 常用策略

1. ATR 止损：入场后以 1.5 到 3 倍 ATR 设置移动止损。
2. 波动过滤：ATR 太低时不做突破，防止假突破。
3. 仓位控制：ATR 越高，单笔仓位越低。
4. SuperTrend：SuperTrend 本身依赖 ATR。

### 局限

ATR 只表示波动大小，不表示方向。高 ATR 既可能是上涨加速，也可能是下跌恐慌。

### 项目建议

重要程度：★★★★☆

V1 建议实现，尤其为 V2 策略、止损和风险控制打基础。

## 10. Volume / Volume MA 成交量与成交量均线

### 解释

成交量表示某个周期内的交易活跃程度。成交量均线则用于平滑观察最近成交量是否显著放大或萎缩。

### 原理

```text
Volume MA = 最近 N 个周期成交量平均值
```

### 用途

1. 判断价格变动是否有资金参与。
2. 过滤无量突破。
3. 识别活跃度变化。
4. 辅助判断趋势是否健康。

### 历史渊源

量价分析是技术分析中的基础思想，早于大多数现代指标。OBV、成交量均线、量价背离等都建立在“价格变化需要成交量确认”的思路之上。

### 常用策略

1. 放量突破：价格突破关键均线或区间，同时成交量超过均量。
2. 缩量回调：上升趋势中价格回调但成交量萎缩，可能只是短期整理。
3. 放量下跌：跌破支撑并放量，警惕趋势转弱。

### 局限

加密货币不同交易所成交量口径不完全一致。V1 只从 Binance 获取数据时，成交量反映的是 Binance 市场，不代表全市场。

### 项目建议

重要程度：★★★★★

V1 必做。列表页至少展示成交量或成交额，以及是否放量。

## 11. Volume Spike 成交量突增

### 解释

Volume Spike 指当前成交量显著高于近期平均成交量，表示市场活跃度突然上升。

### 原理

常见计算方式：

```text
volume_spike_ratio = 当前成交量 / 成交量均线
```

示例阈值：

```text
ratio >= 2.0：明显放量
ratio >= 3.0：异常放量
```

### 用途

1. 突破确认。
2. 异常波动提醒。
3. 辅助识别洗盘、出货或趋势启动。

### 历史渊源

Volume Spike 不是某位作者定义的单一经典指标，而是量价分析中的通用概念。它通常和突破、背离、新闻冲击或流动性事件一起使用。

### 常用策略

1. 放量上破区间：观察趋势启动。
2. 放量跌破支撑：警惕风险释放。
3. 高位放量滞涨：警惕派发。
4. 低位放量止跌：观察反转候选。

### 局限

放量本身不代表方向，需要结合价格位置、K 线结构和趋势背景。

### 项目建议

重要程度：★★★★☆

V1 必做。列表页适合用简洁标签展示，如 `2.4x`、`放量`、`异常`。

## 12. OBV 能量潮

### 解释

OBV 是 On-Balance Volume，中文常称能量潮。它将成交量按价格涨跌方向累计起来，用于观察成交量是否支持当前价格趋势。

### 原理

```text
如果今日收盘价 > 昨日收盘价：OBV = 昨日 OBV + 今日成交量
如果今日收盘价 < 昨日收盘价：OBV = 昨日 OBV - 今日成交量
如果今日收盘价 = 昨日收盘价：OBV = 昨日 OBV
```

### 用途

1. 判断量价是否同步。
2. 观察 OBV 与价格的背离。
3. 辅助确认趋势强弱。

### 历史渊源

OBV 由 Joseph Granville 在 1963 年出版的 *Granville's New Key to Stock Market Profits* 中推广。部分资料也提到早期类似方法曾被称为 continuous volume。

### 常用策略

1. 价格上涨、OBV 同步上涨：趋势较健康。
2. 价格上涨、OBV 不创新高：警惕顶背离。
3. 价格下跌、OBV 不创新低：观察底部吸筹可能。
4. OBV 突破前高：辅助确认资金流入。

### 局限

OBV 对单根异常成交量敏感。加密货币 24 小时交易且跨交易所分散，单一交易所 OBV 需要谨慎解释。

### 项目建议

重要程度：★★★★☆

V1 可作为建议指标，V2 策略中适合做趋势确认和背离过滤。

## 13. Stochastic 随机指标

### 解释

Stochastic Oscillator 中文常称随机指标。它衡量当前收盘价在最近 N 个周期最高价和最低价区间中的相对位置。

### 原理

```text
%K = (Close - Lowest Low) / (Highest High - Lowest Low) * 100
%D = %K 的移动平均
```

常见阈值：

```text
> 80：可能超买
< 20：可能超卖
```

### 用途

1. 震荡行情中寻找短线拐点。
2. 判断收盘价是否接近近期区间顶部或底部。
3. 观察 %K 与 %D 交叉。

### 历史渊源

Stochastic 通常归功于 George Lane，资料普遍认为其在 1950 年代提出，并在 1980 年代进一步流行。Britannica Money 也将其描述为用于识别超买超卖和动量变化的指标。

### 常用策略

1. %K 上穿 %D：短线偏多。
2. %K 下穿 %D：短线偏空。
3. 低位金叉：在 20 以下发生金叉，观察反弹。
4. 高位死叉：在 80 以上发生死叉，观察回落。

### 局限

强趋势中 Stochastic 可长时间停留在高位或低位，逆势使用容易过早入场。

### 项目建议

重要程度：★★★☆☆

V1 可延后。适合详情页辅助观察，不建议作为列表页核心指标。

## 14. CCI 商品通道指数

### 解释

CCI 是 Commodity Channel Index，中文常称商品通道指数。它衡量当前典型价格相对其平均水平的偏离程度。

### 原理

典型价格：

```text
TP = (High + Low + Close) / 3
```

CCI：

```text
CCI = (TP - SMA(TP)) / (0.015 * Mean Deviation)
```

### 用途

1. 识别周期性拐点。
2. 判断价格是否远离均值。
3. 辅助发现新趋势或极端状态。

### 历史渊源

CCI 由 Donald Lambert 在 1980 年提出，最初用于识别商品市场的周期性转折。StockCharts 和 Linn Software 都将其描述为可用于新趋势识别和极端状态提示的指标。

### 常用策略

1. CCI 上穿 +100：可能进入强势区。
2. CCI 下穿 -100：可能进入弱势区。
3. CCI 从极端区回落：观察反转。
4. CCI 与价格背离：观察趋势衰竭。

### 局限

CCI 阈值不是固定真理，不同币种和周期可能需要不同参数。

### 项目建议

重要程度：★★★☆☆

V1 可延后。V2 中适合作为周期拐点和极端偏离过滤器。

## 15. SuperTrend

### 解释

SuperTrend 是趋势跟随指标，通常绘制在价格上方或下方。它依赖 ATR 衡量波动，并用价格是否突破动态线来判断趋势方向。

### 原理

常见核心结构：

```text
Basic Upper Band = (High + Low) / 2 + Multiplier * ATR
Basic Lower Band = (High + Low) / 2 - Multiplier * ATR
```

然后根据价格与上下轨关系切换趋势状态。

### 用途

1. 判断趋势方向。
2. 作为移动止损线。
3. 过滤震荡行情中的入场。
4. 用于趋势跟随策略。

### 历史渊源

SuperTrend 常被归功于 Olivier Seban，资料中常见说法是其在 2000 年代后期提出或推广。相比 RSI、MACD、ATR 等经典指标，SuperTrend 的历史更短，资料来源也相对分散。

### 常用策略

1. 价格上破 SuperTrend 线：趋势转多。
2. 价格下破 SuperTrend 线：趋势转空。
3. 顺 SuperTrend 方向交易，反向信号止损或离场。
4. 与 EMA、MACD、ATR 联合过滤，减少震荡误报。

### 局限

SuperTrend 在震荡行情中容易反复翻转；参数选择会明显影响结果。

### 项目建议

重要程度：★★★★☆

V1 可不优先，V2 策略启动时建议加入。它适合做趋势方向和移动止损，不适合作为唯一信号。

## 16. 指标组合建议

### 16.1 V1 页面核心组合

V1 列表页建议优先展示：

| 展示项 | 目的 |
| --- | --- |
| 最新价格 | 基础行情 |
| 涨跌幅 | 快速识别强弱 |
| 成交量 / 成交额 | 判断活跃度 |
| 周/月/季/4月/半年均线状态 | 判断多周期趋势结构 |
| RSI | 判断动量和超买超卖 |
| MACD 状态 | 判断趋势动量 |
| 布林带位置 | 判断波动区间 |
| 放量倍数 | 判断是否有异常成交 |

### 16.2 V1 详情页核心组合

详情页建议分层展示：

1. 主图：K 线 + SMA/EMA + Bollinger Bands。
2. 成交量区：Volume + Volume MA。
3. 动量区：MACD。
4. 震荡区：RSI。
5. 可选区：ATR、OBV、CCI、Stochastic。

### 16.3 V2 策略组合示例

趋势跟随：

```text
价格站上月线和季度均线
MACD 位于零轴上方
RSI 处于 45-70
成交量高于均量
```

震荡反弹：

```text
价格接近布林带下轨
RSI < 30 后回升
成交量放大
MACD 柱状图收缩
```

突破策略：

```text
布林带挤压后向上突破
Volume Spike >= 2
价格站上周/月均线
MACD 金叉或柱状图转正
```

风险规避：

```text
价格跌破季度/半年均线
MACD 死叉
RSI 跌破 40
放量下跌
```

## 18. 资料依据

关键事实依据：

1. Moving Average、SMA/EMA、Trend Following：Britannica Money 对移动平均、趋势跟随和 SMA/EMA 差异的说明。
2. MACD：Britannica Money 和多方资料均指向 Gerald Appel 在 1970 年代后期开发。
3. RSI、ATR：J. Welles Wilder Jr. 的 1978 年 *New Concepts in Technical Trading Systems*，以及 Google Books / R TTR 文档交叉确认。
4. Bollinger Bands：Britannica Money 与 Bollinger 相关资料均指向 John Bollinger 在 1980 年代早期提出。
5. Stochastic：Britannica Money 和相关资料指向 George Lane 在 1950 年代开发。
6. CCI：StockCharts ChartSchool 与 Linn Software 均指向 Donald Lambert 在 1980 年提出。
7. OBV：资料普遍指向 Joseph Granville 在 1963 年出版物中推广。
8. SuperTrend：公开资料多指向 Olivier Seban，但历史资料集中度低于 Wilder、Appel、Bollinger 等经典指标，因此本文标注为“常被归功于”。
