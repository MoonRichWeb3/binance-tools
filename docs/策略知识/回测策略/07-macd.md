# MACD 趋势策略

## 一句话说明

MACD 用来判断趋势有没有开始变强，或者有没有开始转弱。它像是在看“快线”和“慢线”谁跑得更快。

## 创始人 / 来源

MACD 由 Gerald Appel 在 1970 年代提出。他是一位投资经理和技术分析师，想用均线变化来更早发现趋势转折。

后来 Thomas Aspray 又加入了柱状图的观察方法，让交易者更容易看到动能变强还是变弱。

## 小白怎么理解

可以把 MACD 想成两辆车：

- 快车：反应更快，代表短期价格变化。
- 慢车：反应更慢，代表长期价格变化。

如果快车从下面超过慢车，说明短期上涨力量变强，可能是买入信号。

如果快车从上面跌破慢车，说明短期上涨力量变弱，可能是卖出信号。

MACD 不只看涨跌，还看“涨跌的力气”有没有变强。

## 计算公式

```text
DIF = EMA(12) - EMA(26)
DEA = DIF 的 9 周期 EMA
MACD 柱 = DIF - DEA
```

白话翻译：

- EMA(12)：更看重近期价格的短期均线。
- EMA(26)：更平稳的长期均线。
- DIF：短期和长期之间的距离。
- DEA：DIF 的平滑线。
- 柱子变长：动能变强。
- 柱子变短：动能变弱。

## 当前回测规则

本项目里的 MACD 策略大致按这个逻辑运行：

- DIF 从下往上穿过 DEA，买入。
- DIF 从上往下跌破 DEA，卖出。
- 可以设置单次买入金额和止损比例。

默认参数：

```text
快线周期：12
慢线周期：26
信号线周期：9
单次买入金额：100
止损比例：8%
```

## 适合什么行情

- 趋势刚开始形成。
- 一段震荡后准备突破。
- 中线行情，不追求最低点买入。

## 不适合什么行情

- 很窄的震荡行情，容易频繁买卖。
- 暴涨暴跌行情，信号可能慢半拍。
- 手续费很高的环境。

## 历史故事

MACD 能流行几十年，是因为它把“趋势”和“动能”合在一起看。很多交易者用它来避免只盯价格本身。

但 MACD 的缺点也很明显：它是根据历史价格算出来的，所以一定会有滞后。它更像一个趋势确认工具，不适合当成预测神器。
