# Martingale 马丁策略

## 创始人 / 来源

Martingale 起源于概率和博彩领域，没有单一交易领域创始人。其经典思想是亏损后加倍投入，用下一次盈利覆盖之前亏损。

## 核心原理

价格下跌时按倍数加仓，降低平均持仓成本。只要价格出现足够反弹，就可以达到止盈条件并清仓。

## 计算公式

```text
第 n 层加仓金额 = 首单金额 * 加仓倍数^(n - 1)
平均成本 = 总投入成本 / 持仓数量
止盈价 = 平均成本 * (1 + 止盈% / 100)
加仓价 = 平均成本 * (1 - 加仓跌幅% / 100)
```

## 当前回测规则

- 参数：加仓跌幅%、止盈%、最大层数、首单资金%、加仓倍数。
- 买入：无仓时开首单。
- 加仓：价格低于加仓价且层数未超过上限。
- 卖出：价格达到平均成本上方的止盈比例。

## 适合行情

- 宽幅震荡。
- 有反复反弹的市场。
- 严格限制最大层数和资金暴露。

## 不适合行情

- 单边下跌。
- 高杠杆。
- 流动性突然消失。

## 历史故事

马丁格尔思想在博彩中很有名，因为它看起来“只要有足够资金就能赢”。交易里的问题也正是在这里：资金并不是无限的。马丁策略通常胜率高，但真正危险的是少数极端行情会吞掉大量之前收益。
