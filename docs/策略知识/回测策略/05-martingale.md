# Martingale 马丁策略

## 一句话说明

价格跌了就继续加仓，把平均买入价降下来；只要价格反弹一点，就尝试整体止盈离场。

## 创始人 / 来源

Martingale 起源于概率和博彩领域，没有单一交易领域创始人。经典思想是亏了以后加倍投入，试图用下一次盈利覆盖前面亏损。

## 小白怎么理解

假设你第一次买贵了，价格跌了。马丁策略不是马上认输，而是继续买，让平均成本变低。

例子：

```text
第一次 100 买
第二次 90 买
平均成本就低于 100
价格不用涨回 100，也可能回本
```

问题是：如果价格一直跌，就会越买越多。

## 计算公式

```text
第 n 层加仓金额 = 首单金额 * 加仓倍数^(n - 1)
平均成本 = 总投入成本 / 持仓数量
止盈价 = 平均成本 * (1 + 止盈% / 100)
加仓价 = 平均成本 * (1 - 加仓跌幅% / 100)
```

## 当前回测规则

- 参数：加仓跌幅%、止盈%、最大层数、首单资金%、加仓倍数。
- 买入：无仓时开首单。
- 加仓：价格低于加仓价且层数未超过上限。
- 卖出：价格达到平均成本上方的止盈比例。

## 适合什么行情

- 宽幅震荡。
- 经常有反弹。
- 明确限制最大层数和最大投入资金。

## 不适合什么行情

- 单边下跌。
- 高杠杆。
- 容易暴跌的新币、小币。

## 历史故事

马丁格尔在博彩里很有名，因为它给人一种“只要资金够多，总能赢回来”的感觉。但现实里资金不是无限的。交易中的马丁也是一样：大多数时候看起来胜率高，真正危险的是少数极端下跌。
