# CCI 商品通道指数策略

## 创始人 / 来源

CCI 由 Donald Lambert 在 1980 年提出，最初用于商品市场周期性判断。

## 核心原理

CCI 衡量典型价格相对其均值的偏离程度。数值越低说明价格向下偏离越大，数值越高说明价格向上偏离越大。

## 计算公式

```text
TP = (High + Low + Close) / 3
SMA_TP = SMA(TP, N)
MeanDeviation = mean(abs(TP - SMA_TP))
CCI = (TP - SMA_TP) / (0.015 * MeanDeviation)

常用区间：
CCI < -100：偏弱或超卖
CCI > +100：偏强或超买
```

## 当前回测规则

- 参数：CCI 周期、超卖、超买、单次仓位%、止损%。
- 买入：CCI 小于等于超卖阈值。
- 卖出：CCI 大于等于超买阈值，或触发止损。

## 适合行情

- 周期性震荡。
- 极端偏离后的反弹。
- 商品属性或轮动明显的资产。

## 不适合行情

- 单边趋势。
- 高波动小币种。
- 阈值没有重新适配的币种。

## 历史故事

Lambert 最初设计 CCI 是为了发现商品市场的周期性拐点。后来交易者发现，只要市场存在“价格围绕均值波动”的特征，CCI 就能用来观察偏离程度。
