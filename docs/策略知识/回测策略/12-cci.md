# CCI 顺势策略

## 一句话说明

CCI 用来判断价格是不是明显偏离了正常水平。它像是在看价格有没有突然跑得太远。

## 创始人 / 来源

CCI 由 Donald Lambert 在 1980 年提出。它最早用于商品期货市场，后来被股票、外汇和加密货币交易者广泛使用。

CCI 的英文是 Commodity Channel Index，中文常叫“商品通道指数”。

## 小白怎么理解

价格多数时候会围绕平均水平上下波动。

如果价格明显高于平常水平，CCI 会变高。

如果价格明显低于平常水平，CCI 会变低。

常见理解：

- CCI 高于 100：市场可能进入强势。
- CCI 低于 -100：市场可能进入弱势，或者跌得比较深。

CCI 和 RSI 不太一样。RSI 更像情绪温度计，CCI 更像“偏离平均水平的距离”。

## 计算公式

```text
典型价格 = (最高价 + 最低价 + 收盘价) / 3
CCI = (典型价格 - N 周期平均典型价格) / (0.015 × 平均偏差)
```

白话翻译：

- 先算一根 K 线的代表价格。
- 再看这个价格离最近平均水平有多远。
- 离得越远，CCI 的绝对值越大。

## 当前回测规则

本项目里的 CCI 策略大致按这个逻辑运行：

- CCI 向上突破强势阈值时，买入。
- CCI 跌回或转弱时，卖出。
- 可以设置单次买入金额和止损比例。

默认参数：

```text
周期：20
低位阈值：-100
高位阈值：100
单次买入金额：100
止损比例：8%
```

## 适合什么行情

- 趋势刚变强。
- 价格明显脱离横盘区间。
- 波动节奏比较清楚的市场。

## 不适合什么行情

- 窄幅震荡，容易反复触发。
- 假突破很多的市场。
- 数据跳动异常的小币种。

## 历史故事

CCI 最早不是专门为股票设计的，而是为周期性很强的商品市场设计的。商品价格常常有季节性和周期性，所以交易者需要判断价格是否偏离正常范围。

后来大家发现，这个思想也适合其他市场：不管是什么资产，只要价格会围绕平均水平波动，就可以用 CCI 做参考。
