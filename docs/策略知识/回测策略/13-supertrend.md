# SuperTrend 趋势跟踪策略

## 一句话说明

SuperTrend 用一条会跟随价格移动的线，帮助判断当前是上涨趋势还是下跌趋势。它像一个“趋势方向牌”。

## 创始人 / 来源

SuperTrend 的具体作者在不同资料里说法不完全一致，常见说法会提到 Olivier Seban。它真正流行起来，是因为很多交易软件和脚本平台把它做成了简单好用的趋势指标。

它的核心思想来自两个东西：

- 当前价格方向。
- ATR，也就是市场最近波动有多大。

## 小白怎么理解

可以把 SuperTrend 想成价格旁边的一条保护线：

- 价格在 SuperTrend 线上方：更偏上涨趋势。
- 价格在 SuperTrend 线下方：更偏下跌趋势。

这条线不是固定的。市场波动越大，它离价格越远，避免一点小波动就误判。

市场波动越小，它离价格越近，信号会更敏感。

## 计算公式

SuperTrend 会先计算 ATR，然后根据 ATR 生成上下轨。

```text
中间价 = (最高价 + 最低价) / 2
上轨 = 中间价 + 倍数 × ATR
下轨 = 中间价 - 倍数 × ATR
```

白话翻译：

- ATR 可以理解成“最近价格平均晃动幅度”。
- 倍数越大，线越远，信号越稳但更慢。
- 倍数越小，线越近，信号更快但更容易误判。

## 当前回测规则

本项目里的 SuperTrend 策略大致按这个逻辑运行：

- 趋势从空头切到多头时，买入。
- 趋势从多头切到空头时，卖出。
- 可以设置单次买入金额和止损比例。

默认参数：

```text
ATR 周期：10
ATR 倍数：3
单次买入金额：100
止损比例：8%
```

## 适合什么行情

- 趋势明显的行情。
- 想少做交易，只抓大方向。
- 中短线趋势跟踪。

## 不适合什么行情

- 横盘震荡，容易来回翻方向。
- 小周期噪音很多的行情。
- 想精准抄底逃顶的交易方式。

## 历史故事

SuperTrend 流行的原因是显示非常直观：多数时候你只需要看价格在线上还是线下。

它不像有些指标要同时看很多线，所以很适合做趋势过滤。但它也不是万能的。震荡行情里，它可能频繁反转，导致连续小亏。因此更适合配合市场筛选和仓位控制一起使用。
