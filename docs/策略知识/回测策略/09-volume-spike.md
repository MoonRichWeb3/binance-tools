# 放量突破策略

## 一句话说明

放量突破策略关注“价格上涨时成交量是不是突然变大”。如果涨的时候很多人一起成交，说明这次上涨可能更有力量。

## 创始人 / 来源

放量突破不是某一个人单独发明的策略，而是技术分析里非常早就存在的交易思想。

它的核心来自一句老话：价格上涨要有成交量配合。没有成交量的上涨，可能只是短暂拉一下。

## 小白怎么理解

可以把成交量想成围观和参与的人数：

- 价格涨，但成交量很小：可能只是少数人在推。
- 价格涨，同时成交量明显放大：说明更多资金参与。
- 价格突破前高，同时成交量放大：更像真正突破。

这个策略不是只看价格涨，而是看“涨的时候有没有人跟”。

## 计算公式

```text
平均成交量 = 最近 N 根 K 线成交量平均值
放量倍数 = 当前成交量 / 平均成交量
```

白话翻译：

- 如果最近平均每天成交 100 万。
- 今天成交 250 万。
- 那今天就是 2.5 倍放量。

## 当前回测规则

本项目里的放量突破策略大致按这个逻辑运行：

- 当前价格上涨。
- 当前成交量明显高于过去平均成交量。
- 满足条件后买入。
- 后续用止盈、止损或信号转弱来退出。

默认参数：

```text
成交量周期：20
突破周期：20
放量倍数：2
单次买入金额：100
止损比例：8%
```

## 适合什么行情

- 横盘后突然突破。
- 热点币种启动初期。
- 成交量明显增加的行情。

## 不适合什么行情

- 假突破很多的震荡市场。
- 小币种被短暂拉盘。
- 成交量数据不稳定的交易对。

## 历史故事

“量价配合”是技术分析里最基础也最持久的观念之一。很多经典交易书都会强调：真正的趋势通常需要成交量确认。

但放量不等于一定上涨。有时主力也会利用放量制造假突破，所以这个策略最好结合止损和趋势过滤一起使用。
