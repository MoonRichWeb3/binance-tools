# 布林带策略

## 一句话说明

布林带用三条线把价格包起来，帮助你判断价格是不是跑得太远。它像给价格画了一个“正常活动范围”。

## 创始人 / 来源

布林带由 John Bollinger 在 1980 年代提出。他是一位技术分析师，希望用价格波动本身来动态调整上下边界。

布林带的英文是 Bollinger Bands，所以很多交易软件里会写成 BOLL 或 BBands。

## 小白怎么理解

布林带有三条线：

- 中轨：价格的平均线。
- 上轨：价格偏高的位置。
- 下轨：价格偏低的位置。

可以把它想成一条路：

- 中轨是路中间。
- 上轨是路右边界。
- 下轨是路左边界。

价格靠近下轨，说明最近跌得比较靠下，可能有反弹机会。

价格靠近上轨，说明最近涨得比较靠上，可能有回调风险。

## 计算公式

```text
中轨 = N 周期移动平均线
上轨 = 中轨 + K × 标准差
下轨 = 中轨 - K × 标准差
```

白话翻译：

- N 常用 20，表示看最近 20 根 K 线。
- 标准差可以理解成“价格最近晃动得有多厉害”。
- 晃动越大，布林带越宽。
- 晃动越小，布林带越窄。

## 当前回测规则

本项目里的布林带策略大致按这个逻辑运行：

- 价格接近或跌破下轨时，买入。
- 价格回到中轨或接近上轨时，卖出。
- 可以设置单次买入金额和止损比例。

默认参数：

```text
周期：20
倍数：2
单次买入金额：100
止损比例：8%
```

## 适合什么行情

- 区间震荡。
- 价格经常回到平均值附近。
- 波动有节奏的行情。

## 不适合什么行情

- 强趋势行情，价格可能一直贴着上轨或下轨走。
- 突发消息行情，边界会被快速打穿。
- 流动性差的小币种。

## 历史故事

布林带最有价值的地方是“边界会变”。传统固定百分比边界太死板，而布林带会根据市场波动自动变宽或变窄。

很多新手容易误解：碰到上轨就卖，碰到下轨就买。实际更稳妥的理解是：布林带提醒价格已经偏离平常范围，但是否交易还要看趋势、成交量和风险。
