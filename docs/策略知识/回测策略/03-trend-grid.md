# Trend Grid 趋势过滤网格策略

## 创始人 / 来源

Trend Grid 是网格交易的改良思路，没有单一创始人。它来自交易者对普通网格风险的修正：只在趋势没有破坏时执行网格。

## 核心原理

普通网格在下跌趋势中会持续接货。Trend Grid 加入趋势 EMA 过滤，价格在趋势线上方才允许买入；跌破趋势线并触发止损时清仓。

## 计算公式

```text
EMA_today = Close_today * alpha + EMA_yesterday * (1 - alpha)
alpha = 2 / (N + 1)

网格步长 = (网格上限 - 网格下限) / 网格数量

允许买入：
当前价格 >= 趋势 EMA

止损：
当前价格 < 趋势 EMA * (1 - 止损% / 100)
```

## 当前回测规则

- 参数：网格下限、网格上限、网格数量、趋势 EMA、止损%。
- 买入：价格下穿网格，且价格在趋势 EMA 上方。
- 卖出：价格上穿网格，或跌破趋势 EMA 后触发止损。

## 适合行情

- 偏多震荡。
- 趋势向上但经常回调。
- 希望减少普通网格单边下跌风险。

## 不适合行情

- 快速 V 形反转。
- EMA 频繁被刺穿的高噪声行情。
- 强单边上涨，网格可能过早卖出。

## 历史故事

很多网格交易者在经历单边下跌后，会自然加入趋势过滤。Trend Grid 的故事就是“从只看波动，升级为先判断市场环境”。它不是经典学院派指标，而是实战风控经验的产物。
