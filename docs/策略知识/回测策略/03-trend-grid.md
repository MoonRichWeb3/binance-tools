# Trend Grid 趋势过滤网格策略

## 一句话说明

这是“带安全开关的网格”。只有价格还在趋势线上方时才允许低买高卖；如果趋势坏了，就停止接货并止损。

## 创始人 / 来源

Trend Grid 没有单一创始人。它是普通网格交易者在实战中总结出的改良版：别在明显下跌趋势里一直买。

## 小白怎么理解

普通网格的问题是：价格一直跌，它也一直买。Trend Grid 会先问一句：

```text
现在大方向还好吗？
```

如果价格还在趋势 EMA 上方，说明大方向暂时没坏，可以做网格；如果跌破趋势线太多，就认为情况不对，先离场。

## 计算公式

```text
EMA_today = Close_today * alpha + EMA_yesterday * (1 - alpha)
alpha = 2 / (N + 1)

允许买入：
当前价格 >= 趋势 EMA

止损：
当前价格 < 趋势 EMA * (1 - 止损% / 100)
```

白话翻译：EMA 是一条更看重最近价格的平均线。价格在这条线之上，才认为趋势还可以。

## 当前回测规则

- 参数：网格下限、网格上限、网格数量、趋势 EMA、止损%。
- 买入：价格下穿网格，且价格在趋势 EMA 上方。
- 卖出：价格上穿网格，或跌破趋势 EMA 后触发止损。

## 适合什么行情

- 大方向偏上涨，但中途经常回调。
- 偏多震荡。
- 想做网格，但又不想在大跌里一直买。

## 不适合什么行情

- 快速 V 形反转。
- 插针很多，EMA 经常被打穿。
- 强单边上涨，因为网格可能卖得太早。

## 历史故事

很多人第一次用网格时会喜欢它的稳定小收益，直到遇到一次大跌。Trend Grid 就是这个教训后的改良：网格负责赚波动，趋势线负责告诉你什么时候别再接。
