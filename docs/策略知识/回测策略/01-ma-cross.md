# MA Cross 均线交叉策略

## 一句话说明

看两条“平均价格线”：短线代表最近价格，长线代表更久以前的平均价格。短线往上穿过长线，说明最近价格开始变强；短线往下穿过长线，说明最近价格开始变弱。

## 创始人 / 来源

MA Cross 没有单一创始人。移动平均来自统计学里的“取平均、看大方向”，后来被很多趋势交易者用在股票、外汇、商品和加密货币里。

## 小白怎么理解

如果你只看每天价格，会被涨涨跌跌干扰。均线就是把一段时间的价格揉成一条平滑线。

- 短均线：像“最近几天大家愿意出的平均价”。
- 长均线：像“更长一段时间大家愿意出的平均价”。

当短均线跑到长均线上方，说明最近买盘更积极；当短均线跌回长均线下方，说明热度退了。

## 计算公式

```text
SMA(N) = 最近 N 根 K 线收盘价之和 / N

买入：
上一根短均线 <= 长均线
当前短均线 > 长均线

卖出：
上一根短均线 >= 长均线
当前短均线 < 长均线
```

白话翻译：把最近 N 根 K 线的收盘价求平均。短期平均价从长期平均价下面穿到上面，就买；反过来就卖。

## 当前回测规则

- 参数：短均线、长均线。
- 买入：短均线上穿长均线。
- 卖出：短均线下穿长均线。
- 仓位：买入时使用全部可用现金。

## 适合什么行情

- 价格有明显方向，比如一路慢慢上涨。
- 趋势持续时间比较长。
- 日线、4 小时线这类噪声少一点的周期。

## 不适合什么行情

- 横盘震荡。
- 价格来回假突破。
- 新币历史太短，K 线数量不够。

## 历史故事

均线交叉是很多老派交易系统的基础。它不追求买在最低点、卖在最高点，而是帮交易者识别“大方向是不是变了”。很多趋势交易者接受它会慢半拍，因为他们更在意少犯大错，而不是抓每一个小波动。
