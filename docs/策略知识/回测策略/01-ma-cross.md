# MA Cross 均线交叉策略

## 创始人 / 来源

MA Cross 没有单一创始人。移动平均来自统计学中的时间序列平滑方法，后来被道氏理论、趋势跟随交易者和量化系统广泛使用。

## 核心原理

短期均线反映近期价格变化，长期均线反映更慢的趋势方向。短均线上穿长均线说明近期价格强于中期趋势，策略买入；短均线下穿长均线说明动量转弱，策略卖出。

## 计算公式

```text
SMA(N) = 最近 N 根 K 线收盘价之和 / N

买入信号：
上一根 short_SMA <= long_SMA
当前 short_SMA > long_SMA

卖出信号：
上一根 short_SMA >= long_SMA
当前 short_SMA < long_SMA
```

## 当前回测规则

- 参数：短均线、长均线。
- 买入：短均线上穿长均线。
- 卖出：短均线下穿长均线。
- 仓位：买入时使用全部可用现金。

## 适合行情

- 单边趋势明显。
- 趋势持续时间较长。
- 日线、4 小时线等噪声相对低的周期。

## 不适合行情

- 横盘震荡。
- 价格频繁假突破。
- 新币历史太短、K 线不足。

## 历史故事

均线交叉是很多早期机械交易系统的基础。它的优势不是预测顶部或底部，而是让交易者不再凭感觉追涨杀跌，用一条简单规则跟随大方向。经典趋势跟随系统常接受“小亏很多次，抓住少数大趋势”的结果。
