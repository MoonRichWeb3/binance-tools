# RSI 策略

## 创始人 / 来源

RSI 由 J. Welles Wilder Jr. 提出，发表于 1978 年的 *New Concepts in Technical Trading Systems*。

## 核心原理

RSI 衡量一段时间内上涨幅度和下跌幅度的相对强弱。数值低说明短期下跌力量较强，可能进入超卖；数值高说明短期上涨力量较强，可能进入超买。

## 计算公式

```text
RS = 平均上涨幅度 / 平均下跌幅度
RSI = 100 - 100 / (1 + RS)

常用阈值：
RSI < 30：超卖
RSI > 70：超买
```

## 当前回测规则

- 参数：RSI 周期、超卖、单次仓位%、超买、止损%。
- 买入：RSI 小于等于超卖阈值。
- 卖出：RSI 大于等于超买阈值，或触发止损。

## 适合行情

- 震荡反弹。
- 回调后的短线修复。
- 与趋势过滤配合。

## 不适合行情

- 强下跌趋势。
- 连续阴跌。
- RSI 长时间低位钝化。

## 历史故事

Wilder 设计 RSI 的时代，交易者需要用手工或简单计算器处理数据。RSI 的价值在于把涨跌强弱压缩成 0 到 100 的数字，让交易者直观看到市场是否短期过热或过冷。
