# RSI 超买超卖策略

## 一句话说明

RSI 用来判断一个币最近是不是涨得太急，或者跌得太狠。它像一个“情绪温度计”，温度太高可能要降温，温度太低可能会反弹。

## 创始人 / 来源

RSI 由 J. Welles Wilder Jr. 在 1978 年提出。他原本是机械工程师，后来研究市场交易，并写了《New Concepts in Technical Trading Systems》这本书。

Wilder 还提出过 ATR、ADX、抛物线 SAR 等指标，所以他的很多工具现在仍然被交易软件使用。

## 小白怎么理解

可以把 RSI 想成 0 到 100 的分数：

- 分数很高：最近上涨力量很强，很多人已经追进去了。
- 分数很低：最近下跌力量很强，很多人可能已经恐慌卖出。
- 分数在中间：市场暂时没有特别极端。

常见理解：

- RSI 低于 30：可能跌过头了，策略考虑买入。
- RSI 高于 70：可能涨过头了，策略考虑卖出。

这不是说低于 30 一定涨，也不是高于 70 一定跌。它只是提醒你：市场情绪已经比较极端。

## 计算公式

```text
RSI = 100 - 100 / (1 + RS)
RS = 最近 N 根 K 线平均上涨幅度 / 最近 N 根 K 线平均下跌幅度
```

白话翻译：

- 最近涨得多，RSI 就高。
- 最近跌得多，RSI 就低。
- N 通常用 14，也就是看最近 14 根 K 线。

## 当前回测规则

本项目里的 RSI 策略大致按这个逻辑运行：

- 当 RSI 低于买入阈值时，认为市场可能跌过头，买入。
- 当 RSI 高于卖出阈值时，认为市场可能涨过头，卖出。
- 可以设置每次买入金额和止损比例，避免一次投入太多。

默认参数偏保守：

```text
RSI 周期：14
买入阈值：30
卖出阈值：70
单次买入金额：100
止损比例：8%
```

## 适合什么行情

- 价格在区间里来回震荡。
- 大跌后出现短线反弹。
- 大涨后容易回调的行情。

## 不适合什么行情

- 单边大牛市：RSI 可能长期很高，过早卖出。
- 单边大熊市：RSI 可能长期很低，过早买入。
- 突发消息行情：指标会反应慢。

## 历史故事

RSI 最早是为了帮助交易者判断市场是否过热或过冷。它流行的原因不是因为公式复杂，而是因为结果很直观：一个 0 到 100 的数字，新手也能快速看懂。

不过很多亏损也来自误用 RSI。比如有人看到 RSI 低于 30 就马上梭哈，但强下跌里 RSI 可以低很久。更稳妥的做法是把 RSI 当作提醒，再结合趋势、成交量和风险控制一起看。
