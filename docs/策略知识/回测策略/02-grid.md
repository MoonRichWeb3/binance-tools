# Grid 网格策略

## 创始人 / 来源

Grid 网格交易没有公认的单一创始人。它来自做市、区间交易和机械交易思想，后来在外汇、商品和数字资产市场被广泛使用。

## 核心原理

把价格区间切成多个网格。价格下跌穿过网格线时买入，价格上涨穿过网格线时卖出，尝试从反复波动中积累价差收益。

## 计算公式

```text
网格步长 = (网格上限 - 网格下限) / 网格数量
当前网格 = floor((当前价格 - 网格下限) / 网格步长)

当前网格 < 上一网格：买入
当前网格 > 上一网格：卖出
```

## 当前回测规则

- 参数：网格下限、网格上限、网格数量。
- 买入：价格进入更低网格。
- 卖出：价格进入更高网格。
- 每格订单资金约为初始资金 / 网格数量。

## 适合行情

- 箱体震荡。
- 价格在较稳定区间来回波动。
- 流动性好、滑点低的交易对。

## 不适合行情

- 单边下跌。
- 单边上涨。
- 区间上下限判断错误。

## 历史故事

网格策略流行的原因是它不需要预测未来方向，只需要价格波动。很多交易者最初用手工挂单实现网格，后来交易所和量化工具把它自动化。它看起来稳定，但真正的风险来自“价格不再回到网格区间”。
