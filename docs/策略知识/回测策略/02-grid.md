# Grid 网格策略

## 一句话说明

把价格区间切成很多格子，跌到下一格就买一点，涨到上一格就卖一点，靠来回波动赚差价。

## 创始人 / 来源

Grid 网格交易没有公认的单一创始人。它来自做市、挂单和区间交易思想，后来在外汇、商品和数字资产里很常见。

## 小白怎么理解

想象价格在一个楼梯上上下走：

- 往下走一格，策略觉得“便宜了一点”，买一点。
- 往上走一格，策略觉得“贵了一点”，卖一点。

它不判断未来涨还是跌，只希望价格在区间内反复波动。

## 计算公式

```text
网格步长 = (网格上限 - 网格下限) / 网格数量
当前网格 = floor((当前价格 - 网格下限) / 网格步长)

当前网格 < 上一网格：买入
当前网格 > 上一网格：卖出
```

白话翻译：先规定一个价格范围，再切成很多格。价格跌进更低格子就买，涨进更高格子就卖。

## 当前回测规则

- 参数：网格下限、网格上限、网格数量。
- 买入：价格进入更低网格。
- 卖出：价格进入更高网格。
- 每格订单资金约为初始资金 / 网格数量。

## 适合什么行情

- 价格在一个范围里来回震荡。
- 没有明显单边趋势。
- 流动性好、买卖价差小的币。

## 不适合什么行情

- 一直下跌，会越买越多。
- 一直上涨，会太早卖掉。
- 你设置的上下限不符合真实行情。

## 历史故事

网格交易很受自动化交易者欢迎，因为它简单、机械、容易执行。很多人最早是手工挂一排买单和卖单，后来交易所把它做成自动网格工具。它看起来稳定，但真正危险的是价格离开网格区间后不回来。
