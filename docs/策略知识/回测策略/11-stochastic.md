# KDJ / 随机指标策略

## 一句话说明

随机指标用来判断价格在最近一段范围里，处于偏高还是偏低的位置。它像是在问：现在价格是在这段时间的顶部附近，还是底部附近？

## 创始人 / 来源

随机指标由 George Lane 在 1950 年代推广。它的英文是 Stochastic Oscillator，很多中文软件里会显示为 KDJ。

KDJ 是在随机指标基础上增加了 J 线，让信号更敏感。

## 小白怎么理解

假设最近 14 天价格最低是 90，最高是 110。

如果现在价格是 108，说明它已经接近这段时间高位。

如果现在价格是 92，说明它已经接近这段时间低位。

随机指标就是把这个位置转换成 0 到 100 的分数：

- 分数高：价格在最近区间高位。
- 分数低：价格在最近区间低位。

## 计算公式

```text
%K = (当前收盘价 - N 周期最低价) / (N 周期最高价 - N 周期最低价) × 100
%D = %K 的移动平均
```

白话翻译：

- K 线更敏感，变化更快。
- D 线更平滑，变化更慢。
- K 从下往上穿 D，可能转强。
- K 从上往下跌破 D，可能转弱。

## 当前回测规则

本项目里的随机指标策略大致按这个逻辑运行：

- 指标处于低位并出现转强信号，买入。
- 指标处于高位并出现转弱信号，卖出。
- 可以设置单次买入金额。

默认参数：

```text
周期：14
平滑周期：3
低位阈值：20
高位阈值：80
单次买入金额：100
```

## 适合什么行情

- 区间震荡。
- 短线反弹和回调。
- 有明显高低区间的市场。

## 不适合什么行情

- 强趋势行情，指标可能一直高或一直低。
- 波动特别剧烈的行情。
- 手续费很高的频繁交易环境。

## 历史故事

George Lane 认为，价格在真正反转前，收盘位置经常会先出现变化。比如上涨趋势变弱时，价格可能还在高位，但收盘已经不再靠近最高点。

随机指标因此很适合观察短线转折。不过它太敏感，容易给出很多信号，所以最好配合趋势过滤和止损。
