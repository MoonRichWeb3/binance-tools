# Stochastic 随机指标策略

## 创始人 / 来源

Stochastic Oscillator 通常归功于 George Lane，流行于 1950 年代之后。

## 核心原理

Stochastic 衡量当前收盘价在最近高低点区间中的位置。强势时收盘价常接近高位，弱势时常接近低位。低位金叉可能代表短线反弹，高位死叉可能代表回落。

## 计算公式

```text
%K = (Close - LowestLow) / (HighestHigh - LowestLow) * 100
%D = SMA(%K, D周期)

买入：低位 %K 上穿 %D
卖出：高位 %K 下穿 %D
```

## 当前回测规则

- 参数：K 周期、D 周期、超卖、超买、单次仓位%。
- 买入：低位 %K 上穿 %D。
- 卖出：高位 %K 下穿 %D。

## 适合行情

- 区间震荡。
- 短线超买超卖。
- 反复波动的市场。

## 不适合行情

- 强趋势行情。
- 追涨杀跌阶段。
- 指标长时间钝化。

## 历史故事

George Lane 强调 Stochastic 不是跟随价格本身，而是跟随价格在区间里的位置。它试图回答一个问题：收盘价是在最近区间的强势位置，还是弱势位置？
