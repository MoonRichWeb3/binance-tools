# Turtle 海龟策略

## 一句话说明

价格突破过去一段时间高点就买，跌破规则设定的位置就卖。它不抄底，只追随已经出现的强趋势。

## 创始人 / 来源

海龟交易法由 Richard Dennis 和 William Eckhardt 在 1980 年代实验和推广。

## 小白怎么理解

海龟策略的想法是：

```text
别猜什么时候会涨。
等它真的突破了，再跟上。
```

它像追火车：火车没开动时不跳上去，等它明显启动了再上车。如果火车掉头，就下车。

## 计算公式

```text
入场高点 = 最近 entry_window 根 K 线最高价
退出低点 = 最近 exit_window 根 K 线最低价

TR = max(High - Low, abs(High - PreviousClose), abs(Low - PreviousClose))
ATR = TR 的均值

买入：Close > 入场高点
卖出：Close < 退出低点 或 Close <= 入场价 - ATR * 止损倍数
```

白话翻译：突破最近高点才买；跌破最近低点或跌得超过允许范围就卖。

## 当前回测规则

- 参数：突破周期、退出周期、单次仓位%、ATR 周期、止损 ATR。
- 买入：收盘价突破过去突破周期最高价。
- 卖出：收盘价跌破退出周期最低价，或触发 ATR 止损。

## 适合什么行情

- 强趋势。
- 趋势持续时间长。
- 波动大但方向明确。

## 不适合什么行情

- 横盘震荡。
- 假突破频繁。
- 流动性差、插针多的币。

## 历史故事

Richard Dennis 和 William Eckhardt 曾争论：交易能力是天生的，还是可以训练出来的？他们招募普通人，教一套突破、止损、仓位规则。这批学员被称为“Turtles”。海龟交易法真正出名的不是某个神奇指标，而是纪律和风险控制。
