# OBV 能量潮策略

## 一句话说明

OBV 用成交量来判断资金是在慢慢流入，还是慢慢流出。它像在看市场背后的“资金方向”。

## 创始人 / 来源

OBV 由 Joseph Granville 在 1960 年代提出。他认为成交量经常会比价格更早反映资金意图。

OBV 的英文是 On-Balance Volume，中文常叫“能量潮”。

## 小白怎么理解

OBV 的想法很简单：

- 今天收盘价比昨天高，就把今天成交量加上去。
- 今天收盘价比昨天低，就把今天成交量减掉。
- 如果价格没变，OBV 也不变。

这样累积下来，就能看到资金大方向。

如果价格还没怎么涨，但 OBV 已经一直往上，可能说明资金在提前进场。

如果价格还没怎么跌，但 OBV 已经一直往下，可能说明资金在提前离场。

## 计算公式

```text
如果今日收盘价 > 昨日收盘价：
OBV = 昨日 OBV + 今日成交量

如果今日收盘价 < 昨日收盘价：
OBV = 昨日 OBV - 今日成交量

如果今日收盘价 = 昨日收盘价：
OBV = 昨日 OBV
```

白话翻译：

上涨日的成交量算作资金流入，下跌日的成交量算作资金流出。

## 当前回测规则

本项目里的 OBV 策略大致按这个逻辑运行：

- OBV 趋势向上，并且价格配合，买入。
- OBV 趋势转弱或价格跌破条件，卖出。
- 可以设置单次买入金额和止损比例。

默认参数：

```text
OBV 短周期：20
OBV 长周期：30
单次买入金额：100
止损比例：8%
```

## 适合什么行情

- 资金慢慢进场的行情。
- 价格还没明显启动，但成交量已经变化。
- 中短线趋势判断。

## 不适合什么行情

- 成交量异常跳变的小币种。
- 洗盘频繁的市场。
- 没有明确方向的震荡行情。

## 历史故事

Granville 认为“成交量是价格的燃料”。如果没有资金持续参与，价格上涨往往难以持续。

OBV 的优点是逻辑直观，缺点是容易被异常成交量影响。一次很大的成交量可能让 OBV 变化很明显，所以实际使用时要避免只看单根 K 线。
