# 蜡烛图（K线）基础入门

## 什么是蜡烛图？

蜡烛图（Candlestick Chart），也叫 K线图，是一种记录价格变化的图表方式，广泛应用于：

- 股票
- 外汇
- 加密货币
- 期货

每一根 K 线，代表某个时间周期内的价格变化。

例如：

- 1分钟K线 → 表示1分钟内价格变化
- 1小时K线 → 表示1小时内价格变化
- 日K线 → 表示一天内价格变化

蜡烛图最早起源于日本米市，因此也叫“日本蜡烛图”。

---

# 一根K线包含什么？

一根标准K线由四个价格组成：

| 名称 | 含义 |
|---|---|
| 开盘价（Open） | 周期开始时价格 |
| 收盘价（Close） | 周期结束时价格 |
| 最高价（High） | 周期内最高价格 |
| 最低价（Low） | 周期内最低价格 |

---

# K线结构

```text
      最高价
         │
         │
     ┌───────┐
     │ 实体  │
     └───────┘
         │
         │
      最低价
```

K线由：

- 实体（Body）
- 上影线（Upper Shadow）
- 下影线（Lower Shadow）

组成。

---

# 阳线与阴线

## 阳线（上涨）

当：

```text
收盘价 > 开盘价
```

表示价格上涨。

通常：

- 绿色
- 白色

代表上涨。

```text
开盘价
   ┌────┐
   │    │
   │    │
   └────┘
收盘价
```

---

## 阴线（下跌）

当：

```text
收盘价 < 开盘价
```

表示价格下跌。

通常：

- 红色
- 黑色

代表下跌。

```text
收盘价
   ┌────┐
   │    │
   │    │
   └────┘
开盘价
```

---

# K线的意义

K线反映：

- 多空力量
- 市场情绪
- 买卖压力
- 趋势变化

通过观察K线组合，可以分析：

- 是否上涨
- 是否下跌
- 是否反转
- 是否震荡

---

# 常见K线形态

---

# 1. 锤子线（Hammer）

## 特征

- 下影线很长
- 实体较小
- 上影线很短

```text
   ┌───┐
   │   │
   └───┘
      │
      │
      │
```

## 含义

表示：

- 空头曾大幅打压
- 但多头最终拉回

通常意味着：

```text
可能见底反转
```

---

# 2. 倒锤子线（Inverted Hammer）

```text
      │
      │
   ┌───┐
   │   │
   └───┘
```

## 含义

- 多头尝试反攻
- 空头仍存在压力

可能出现：

```text
底部反转信号
```

---

# 3. 吞没形态（Engulfing）

## 看涨吞没

后一根阳线完全包住前一根阴线。

```text
小阴线 → 大阳线
```

含义：

```text
多头力量增强
```

---

## 看跌吞没

后一根阴线完全包住前一根阳线。

```text
小阳线 → 大阴线
```

含义：

```text
空头力量增强
```

---

# 4. 十字星（Doji）

## 特征

开盘价 ≈ 收盘价

```text
    │
 ───┼───
    │
```

## 含义

表示：

```text
多空平衡
市场犹豫
```

如果出现在高位：

```text
可能下跌
```

如果出现在低位：

```text
可能上涨
```

---

# K线时间周期

不同周期代表不同级别趋势。

| 周期 | 含义 |
|---|---|
| 1m | 超短线 |
| 5m | 短线 |
| 15m | 日内交易 |
| 1H | 波段 |
| 4H | 中期趋势 |
| 1D | 长线趋势 |

---

# K线与趋势

## 上升趋势

特征：

- 高点越来越高
- 低点越来越高

```text
Higher High
Higher Low
```

---

## 下降趋势

特征：

- 高点越来越低
- 低点越来越低

```text
Lower High
Lower Low
```

---

# K线中的市场心理

K线本质是：

```text
人性的可视化
```

例如：

- 长阳线 → 市场贪婪
- 长阴线 → 市场恐慌
- 十字星 → 犹豫
- 锤子线 → 抄底资金进入

---

# K线分析核心原则

## 1. 单根K线意义有限

不要只看一根K线。

要结合：

- 趋势
- 成交量
- 支撑压力
- 多根K线组合

一起分析。

---

## 2. 趋势比形态更重要

```text
顺势交易
永远优先
```

上涨趋势中的看涨形态更有效。

下跌趋势中的看跌形态更有效。

---

## 3. 大周期优先

例如：

- 日线趋势 > 1小时趋势
- 4小时趋势 > 15分钟趋势

---

# K线与成交量

成交量非常重要。

## 放量上涨

```text
上涨更可信
```

---

## 缩量上涨

```text
可能是假突破
```

---

## 放量下跌

```text
恐慌抛售
```

---

# 新手常见错误

## 1. 只看单根K线

错误。

必须结合：

- 趋势
- 成交量
- 结构

---

## 2. 频繁交易

K线不是每次都有效。

市场存在：

- 噪音
- 假突破
- 诱多诱空

---

## 3. 忽略风险管理

再强的K线：

```text
也不是100%正确
```

必须：

- 设置止损
- 控制仓位
- 避免重仓

---

# K线学习路线

建议学习顺序：

```text
1. K线基础
2. 趋势
3. 支撑压力
4. 成交量
5. 均线
6. MACD
7. RSI
8. 交易系统
9. 风险管理
```

---

# 总结

蜡烛图是交易市场最核心的分析工具之一。

它能够帮助交易者：

- 观察趋势
- 判断情绪
- 寻找买卖点
- 分析多空力量

但：

```text
K线不是预测未来
而是理解市场行为
```

真正稳定盈利的核心：

```text
趋势 + 风控 + 纪律
```

而不是单纯依赖某一个K线形态。
